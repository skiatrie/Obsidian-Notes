- use keyword [extend] to achieve inheritance
`public class EmployeeWithTerritory extends Employee`

- you may add your own data fields to the new class:
```java
public class EmployeeWithTerritory extends Employee 
{
	private int territory;
	public int getTerritory()
	{
	return territory
	}
	public int setTerritory(int terr)
	{
	territory = terr;
	}
}
```

instantiate a derived class object:

`EmployeeWithTerritory northernRep = new EmployeeWithTerritory();`
- then you'd be able to use any get field values from parent class
- `northernRep` object has access to methods in parent class as well as it's own newly created ones within `EmployeeWithTerritory`
BUT
- the parent class does NOT have access to it's child's attributes


use the `instanceof` operator to check whether an object is a member or descendant of a class
`northernRep instance of EmployeeWithTerritory`
`northernRep instance of Employee`

- `instanceof` yields `true` if operand on left can be **upcast** to the operand on right
[upcast] - change an object to an object of a class higher in inheritance hierarchy
